My Favorite AI Tools

A list of AI tools I love as a developer.

As a Computer Science student and AI enthusiast, I love testing new AI tools to speed up my work. Here are my current favorites:

  • ChatGPT: My go-to assistant for quick code snippets, explanations, and brainstorming ideas.
  • Midjourney: For generating creative images for project mockups and blog banners.
  • Github Copilot: It helps me write code faster, especially repetitive patterns.
  • Runway ML: For experimenting with AI video and image generation.
